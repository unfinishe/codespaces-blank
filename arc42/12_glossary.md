# 12 Glossary

- **SAF**: Storage Access Framework, Android mechanism for user-selected file/folder access.
- **MVP**: Minimum Viable Product.
- **Folder Schema**: Rule that maps date information to destination folder path.
- **Conflict Resolver**: Logic that renames files if destination filename already exists.
- **Sort Report**: Final run summary (processed, copied/moved, failed, skipped).
- **Feature Parity**: Android implementation matches legacy Python behavior for defined scope.
